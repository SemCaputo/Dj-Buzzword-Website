Reel Big Bottom

Reel Big Bottom is a collection of 240+ Double Bass Loops played by a seasoned session player to 1/2" and 1/4" tape to get that vintage vibe we love. This pack was inspired by classic hip hop tracks sampling upright bass grooves. Think talkin' all that jazz, don't sweat the technique, props over here, sure shot, one love, put it on, the choice is yours... A lot of heavy tracks have used double bass sampled from classic jazz records. We went back to tape through an analogue signal path using various vintage and modern microphones and preamps to get back to that sound.

  • Recorded to 1/4" and 1/2" Analog Tape
